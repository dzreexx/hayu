Multi Payment for Invoices & Bills on single Customer/Vendor

This module is tested in community & Enterprise versions 15.

Features:

  • Create Payment for Invoices & Bills on Single Customer/Vendor
  • Allocate Full Payment, Partial, Write Off in the same Payment Form

  • Procedures :
Create Multi Invoices/Vendor Bills with the same Customer/Vendor
Create Payment
Allocate the Payment with Full, Partial or Write Off Payment
Status Payment for each Invoice
Journal
