ZKY Stock Information Module
=============================

This module adds stock information to quotation lines:

Features:
1. On-Hand Quantity: Shows the total quantity available across all locations
2. Available Quantity: Shows the quantity available in the quotation's warehouse

Both fields are visible in quotation lines without requiring the quotation to be saved first.

Installation:
1. Place the module in your Odoo addons directory
2. Restart your Odoo server
3. Update your apps list from the Apps menu
4. Search for "ZKY Stock Information" and install it
